# Sobre

Repositório criado com o intuito de elaborar e praticar as principais estruturas de dados em Javascript.

# Estrutura de Dados

As estruturas de dados a serem criadas são:
* Algoritmos de Ordenação
  * selection sort
  * insertion sort
  * bubble sort
  * merge sort
  * quick sort

* Algoritmos de Busca
  * binary search
